# Landing App

Marketing/landing frontend built with Next.js.

## Default Port

- `3001`

## Run

```bash
pnpm -C apps/landing dev
pnpm -C apps/landing build
pnpm -C apps/landing start
```

## Runtime Dependencies

- `next`, `react`, `react-dom`
- UI/styling: `tailwind-merge`, `class-variance-authority`, `clsx`, `tw-animate-css`, `shadcn`, `radix-ui`
- Icons and motion: `lucide-react`, `motion`

## Dev Dependencies

- `typescript`, `eslint`, `eslint-config-next`
- `tailwindcss`, `@tailwindcss/postcss`
- `@types/node`, `@types/react`, `@types/react-dom`
